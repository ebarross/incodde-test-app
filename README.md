### Build With

*  Styled Components
*  Axios

## Getting Started

This is an example of how you can run the project locally.

### Installation

1.  Clone the repo
```sh
git clone https://github.com/ebarross/todo-app-mobile.git
```

2.  Install NPM packages
```sh
npm install
```

3. Run the API locally

4.  Create `.env` from `.env.example` inserting your computer's local IP and the port the API is running on
```sh
SERVER_HOST=http://192.168.1.2
SERVER_PORT=3000
```

### Running

```sh
npm run android
```
or
```sh
npm run ios
```
